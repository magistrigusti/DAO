# Dominum Contracts Explorer (tools)

Это отдельный инструмент для визуализации **дерева контрактов** и их связей.

## Запуск (Windows / PowerShell)

Перейти в папку:

- `tools/contracts-explorer/`

Установить зависимости:

- `npm install`

Запустить dev:

- `npm run dev`

Открыть в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## Данные

Контракты и связи описаны в:

- `src/data/contracts.ts`

## Схемы

Отдельная страница со схемами лежит тут:

- `../contracts/index.html`
